PandoraJS

Bienvenue dans ce tutoriel sur comment utiliser PandoraJS. PandoraJS est née dans le but de répondre a un besoin de crée des designs a tuile qui possèdent des animations dynamique tout en étant responsive pour pouvoir s'adapter à tous les écrans des futurs utilisateurs. Cette animation se veut très basique, et il s’agit d’agrandir la zone dans laquelle l’utilisateur est focaliser, que ce soit par un effet de < onclick > ou bien de < hover > tout en gardant le reste du contenu organiser et responsive. Bien sûr l’utilisateur pourra changer de tuiles de focalisation à tout moment et le contenu de la page se redimensionnera parfaitement.

Suivez ce lien pour voir des exemples d’utilisation.

En plus de se vouloir pratique, PandoraJS a l'ambition d'être simple d'utilisation, c'est donc pourquoi c'est un plugin JQuery. C’est pourquoi en 4 étapes vous allez pouvoir avoir vos tuiles dynamiques.

Etape 1 : Récupérer le fichier PandoraJS ainsi que JQuery

Avant de pouvoir utiliser PandoraJS il vous faudra récupérer le fichier pour pouvoir l’inclure dans votre projet.

Pour se faire il suffit de se rendre sur le github du projet : http://github.com/lferrand/pandoraJS

N’oubliez pas que PandoraJS est un plugin JQuery, il faudra aussi récupérer ce fichier : http://jquery.com/download/

Maintenant il faut rajouter ces ficher a votre fichier HTML avec la balise <script>

 

Etape 2 : Votre présentation HTML

 

PandoraJS se veut le plus simple possible d’utilisation, mais on ne vous fournis pas le contenu de vos tuile, il faudra les inclure vous-même et cette partie va servir exactement a ceci.

Pour que PandoraJS fonctionne il faudra respecter un minimum de formating dans votre fichier HTML. Chaque boite de tuile devra être contenue dans une balise <section> avec l’id de votre choix. Dans cette <section> chaque ligne devra être dans une balise <div> qui aura comme nom de class : wrapper. Une fois que vous avez créé vos div wrapper il faut maintenant crée vos tuiles. Chaque tuile sera placée dans une balise <article> qui portera comme nom de class : ligne<numeroDeLaLigne> . Il est important d’inclure le numéro de la ligne car pour l’instant le plugin ne le détecte pas encore automatiquement et que celle-ci soit la première dans votre liste de classe.

Finalement, le but de ce plugin étant d’agrandir la tuile dans laquelle l’utilisateur est focalisé Pandora va nécessiter qu’on passe une classe : current. Il est très important de le placer en  dernier dans vos liste de class.

Mais puisque un exemple parle souvent mieux que des mots, voici a quoi devrait ressembler votre HTML.


<section id="section1">
	<div class="wrapper">
		<article class="ligne1 current">
			TEST ligne 1
			<br/>boite 1
		</article>
	    <article class="ligne1">
			TEST ligne 1
			<br/>boite 2
		</article>
	</div>
	<div class="wrapper">
		<article class="ligne2">
			TEST ligne 2
			<br/>boite 1
		</article>
		<article class="ligne2">
			TEST ligne 2
			<br/>boite 2
		</article>
		<article class="ligne2">
			TEST ligne 2
			<br/>boite 3
		</article>
	</div>
</section>

 

Etape 3 : Votre fichier CSS

Malheureusement pour l’instant PandoraJS ne fournis pas automatiquement tout le CSS nécessaire a son fonctionnement. Il vous faudra donc faire quelques efforts encore avant de pouvoir enfin profiter de ce fabuleux outil.

On va placer quelque paramètre CSS a nos élément HTML qu’on vient de créer.

Il vous faudra d’abord ajouter une width à votre <section> qui contiendra vos tuiles. N’oubliez pas que PandoraJS est responsive donc hésité pas à utiliser une largeur en pourcentage !


#section1{
	width: 100%;
}

Maintenant que vous avez une largeur il faut maintenant dire aux wrapper de prendre toute la largeur de la <section> qui les contient. On veut aussi éviter que le javascript nous créé du overflow par erreur, on va donc mètre un « overflow : hidden ; » et finalement pour que nos ligne soit bien on va lui rajouter un type de display, « display : flex ».


.wrapper {
  	overflow:hidden;
  	width:100%;
  	display: flex;
}

Enfin on passe aux tuiles elle-même. Pour les tuiles elle-même vous êtes plus ou moins libres car il s’agit après tout de votre contenu. Tout ce dont PandoraJS a besoin est que vous fixez l’auteur de la ligne et que vous dites que le display doit être en block. Heureusement ça tombe bien, un peu plus haut nous avons créé une classe ligne<numeroDeLaLigne> qu’on va pouvoir réutiliser dans notre fichier CSS. On aura quelque chose du genre.


.ligne1 {
    height: 200px;
    display: block;
  }
.ligne2 {
    height: 200px;
    display: block;
  }

 Etape 4 : Utiliser le plugin JQuery !

Le meilleur et le plus facile pour la fin. Dans un nouveau fichier séparer que vous allez faire gaffe de bien inclure après le fichier JQuery et du plugin, vous allez pouvoir faire appel au super et magnifique plugin PandoraJS. Pour ce faire il vous suffira d’une simple ligne de code :


$(document).ready(function() {
	$('#section1').pandoraJS(
	{
		'hZoom': 1.3,
		'wZoom': 1.1,
		'type': 'onclick'
	});
});

Comme vous pouvez le voir il a trois paramètres possible qui sont les suivant :
hZoom : qui doit être un chiffre, c’est le ratio de zoom en hauteur.
wZoom : qui doit lui aussi être un chiffre, c’est le ration de zoom en largeur.
type : qui doit cette fois être une chaine de caractères du nom du type de l’animation qu’on souhaite. Pour l’instant il y en a que trois. Le hover, le onclick, et le onclickScale qui permet d’agrandir le texte contenu dans les tuiles en proportion aussi.

-        Si vous laissez le premier vide, alors le ratio du zoom hauteur (hZoom) sera de 150/100

-        Si vous laissez le deuxième vide, alors le ratio du zoom largeur (wZoom) sera de 133.32/100

-        Si vous laissez le troisième vide, alors le type d’animation sera le onclick basique.

Voici un autre exemple en tenant compte de la connaissance de ces paramètres.


$(document).ready(function() {
	$('#section1').pandoraJS(
	{
		'hZoom': 1.6,
		'type': 'onclickScale'
	});
});
